# ComicVault - Front-end

## Sobre o projeto

O ComicVault é uma aplicação SPA (Single Page Application) desenvolvida para organizar coleções de histórias em quadrinhos.

A aplicação consome a API REST desenvolvida em Flask e permite cadastrar, visualizar, editar, excluir e pesquisar HQs.

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

---

## Funcionalidades

- Cadastro de HQs
- Listagem em Cards
- Pesquisa por título
- Filtro por status
- Edição de HQ
- Exclusão de HQ

---

## Instalação

Clone o repositório:

```bash
git clone https://github.com/SEU_USUARIO/ComicVault_Front-end.git
```

Entre na pasta:

```bash
cd ComicVault_Front-end
```

---

## Executando o projeto

Antes de abrir o front-end, certifique-se de que a API esteja em execução.

Abra o arquivo:

```
index.html
```

diretamente no navegador.

A aplicação irá consumir a API em:

```
http://127.0.0.1:5000
```
